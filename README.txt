Introduction
============

The ``gs.cache`` product provides a mechanism to cache values that are not
stored, but are computationally difficult to figure out. Behind the scenes
`the Python redis module`_ is used to store the key-value pairs in `the
redis data structure server`_.

.. _the Python redis module: http://pypi.python.org/pypi/redis/
.. _the redis data structure server: http://redis.io/
